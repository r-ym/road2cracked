# AI and LLMs

Study materials for AI concepts and Large Language Models.

- Supervised, unsupervised, reinforcement learning
- Neural networks and backpropagation
- Optimization algorithms (SGD, Adam, etc.)
- Regularization and generalization

### Deep Learning
- CNNs, RNNs, LSTMs, GRUs
- Attention mechanisms
- Transformer architecture
- Transfer learning and fine-tuning

### Large Language Models
- Model architectures (GPT, BERT, T5, etc.)
- Training techniques and scaling laws
- Prompt engineering
- RAG (Retrieval Augmented Generation)
- Fine-tuning and RLHF

### AI Systems and Infrastructure
- Model serving and deployment
- Inference optimization
- Distributed training
- MLOps and monitoring

## Practical Applications

- Implement basic neural networks from scratch
- Work with popular frameworks (PyTorch, TensorFlow)
- Fine-tune small models
- Build LLM-powered applications
