# Rust Learning

Resources and practice for learning Rust programming language.

## Learning Path

1. **Fundamentals**
   - Ownership and borrowing
   - Lifetimes and references
   - Pattern matching
   - Error handling

2. **Intermediate**
   - Traits and generics
   - Closures and iterators
   - Smart pointers
   - Concurrency

3. **Advanced**
   - Unsafe Rust
   - Macros and metaprogramming
   - Async programming
   - FFI and interop

## Practice Projects

- Implement data structures from C++ in Rust
- Solve LeetCode problems in Rust
- Build small systems/tools
- Concurrent programming exercises
