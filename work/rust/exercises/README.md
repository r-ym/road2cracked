# Rust Exercises

Progressive exercises to build Rust proficiency. Complete them in order.

## Getting Started

Each exercise has:
- Problem description
- Test cases
- Hints (if needed)

Run tests with:
```bash
cargo test
```

## Exercise Progression

### Basics (Week 1)
1. `01_hello` - Basic syntax and printing
2. `02_variables` - Variables and mutability
3. `03_functions` - Function definitions
4. `04_control_flow` - if/else, loops

### Ownership (Week 2) - CRITICAL
5. `05_ownership_basics` - Understanding ownership
6. `06_references` - Borrowing and references
7. `07_slices` - Working with slices
8. `08_strings` - String vs &str

### Structs and Enums (Week 3)
9. `09_structs` - Define and use structs
10. `10_methods` - Implement methods
11. `11_enums` - Enums and pattern matching
12. `12_option_result` - Option and Result types

### Collections (Week 4)
13. `13_vectors` - Vector operations
14. `14_hashmaps` - HashMap usage
15. `15_iterators` - Iterator methods

### Advanced (Week 5+)
16. `16_generics` - Generic functions and structs
17. `17_traits` - Trait definitions and implementations
18. `18_lifetimes` - Lifetime annotations
19. `19_closures` - Closures and functional programming
20. `20_smart_pointers` - Box, Rc, RefCell

## LeetCode-Style Problems

Solve these common interview problems in Rust:
- Two Sum
- Valid Parentheses
- Merge Two Sorted Lists
- Binary Search
- Reverse Linked List
- Maximum Subarray
- Climbing Stairs

Compare your Rust solutions with C++ versions!
