# Interview Preparation Repository

This repository is organized for HFT and AI startup interview preparation, focusing on mid-level to senior software developer positions.

## Structure

### `/content`
Reference materials, documentation, and study resources.

### `/work`
Active practice and learning materials organized by topic:

- **`services/`** - Company-specific interview prep and application tracking
- **`leetcode/`** - LeetCode problem solutions and patterns (C++)
- **`compilers/`** - Compiler design and implementation studies
- **`datastructures/`** - Data structure implementations and analysis (C++)
- **`algorithms/`** - Algorithm implementations and complexity analysis (C++)
- **`rust/`** - Rust language learning and practice
- **`ai-llms/`** - AI and Large Language Model studies

## Goals

- Master C++ fundamentals for HFT interviews
- Build strong algorithmic problem-solving skills
- Learn Rust as a secondary language
- Understand AI/LLM concepts for AI startup roles
- Prepare for system design and compiler-related questions
